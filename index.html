Interactive Campus Map

In Spring 2013 I was working on developing an Interactive Campus Map as part of GEOG790. This involved learning JavaScript, Leaflet and OpenStreetMap. Campus buildings and features were added to OpenStreetMap as part of this project, and the MapQuest Nominatum API and Overpass API were used to access the OSM data. For this project we primarily used the JQuery and Leaflet libraries.

GIS Jobs Map

In Spring 2013 I took the GEOG567 course at NIU, creating a map using Adobe Illustrator. We had to incorporate some GIS layers, I used city and climate data along with major interstates and state boundaries. I also used this shaded relief from Shadedrelief.com as a background. The project involved a lot of digitizing in Illustrator, in fact almost everything other than the relief map!
